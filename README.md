# SNAPFIT

<br>

### 이름 (Name)
**SNAPFIT (스냅핏)** - 추상 전략 보드게임

### 요구사항 (Requirements)
*   **운영체제**: Windows, macOS, Linux 등 OS 무관
*   **브라우저**: 최신 웹 브라우저 (Chrome, Edge, Safari, Firefox 등)
*   **로컬 서버**: 브라우저 보안 정책(CORS) 우회를 위해 `Node.js (npx)` 또는 `Python`이 설치된 환경 권장

### 사용법 (Usage)
1.  **로컬 서버 실행**: 
    터미널(또는 명령 프롬프트)을 열고 프로젝트 폴더로 이동한 뒤, 아래 명령어 중 하나를 실행하여 로컬 서버를 엽니다.
    *   **Python 사용 시**: `python -m http.server 8000` (구버전은 `python -m SimpleHTTPServer`)
    *   **Node.js 사용 시**: `npx serve` 또는 `npx http-server`
2.  **게임 접속**:
    브라우저를 열고 `http://localhost:8000` (또는 터미널에 안내된 포트 주소)로 접속합니다.
    *(참고: 파일 탐색기에서 `index.html`을 직접 실행하면 브라우저 보안 정책에 의해 에셋 파일 로딩이 차단될 수 있습니다.)*
3.  **조작 방법**:
    *   **배치**: 게임 UI의 타일 목록에서 놓을 타일을 누른 뒤, 보드의 원하는 위치를 클릭합니다.
    *   **이동**: 보드 위에 놓인 타일 또는 타일 그룹을 마우스(또는 터치)로 드래그하여 이동합니다.
    *   **회전**: 타일 선택 후 마우스 휠을 굴리거나 키보드 `Q` / `E` 키를 눌러 회전시킵니다.
    *   **장애물 설정 (게임 전)**: 설정 단계에서 원하는 장애물을 선택한 뒤 보드를 클릭하여 배치합니다. 선택된 장애물은 키보드 `Delete` 키로 삭제할 수 있습니다.
    *   **취소**: `Esc` 키를 누르면 현재 진행 중인 배치를 취소합니다.
